hexo部署博客步骤

1.安装node.js （输入node -v检测安装成功与否）
2.使用npm install -g cnpm –registry=https://registry.npm.taobao.org命令安装淘宝镜像（输入cnpm -v检测安装成功与否）
3.使用cnpm install -g hexo-cli命令安装hexo（输入hexo -v检测安装成功与否）
4.创建个人博客文件夹mkdir ybx_blog
5.进入个人博客文件夹输入命令hexo s启动博客
6.浏览器输入localhost:4000查看博客
7.结束运行 输入hexo n“文章标题”创建一篇文章 自行使用方法编写文章
8.hexo clean清理
9.hexo g生成
